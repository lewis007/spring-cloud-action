# Spring Cloud 实战

基于 Spring Boot 2.x 和 Spring Cloud Finchley.Mx 的微服务实战。

## 总体进度

虽说是实战，但还是以系统性学习为主，未在项目中实际使用，不具备项目脚手架作用。

* 服务注册与发现
* 客户端负载均衡
* 熔断器/断路器
* Hystrix 仪表盘
* 网关
* 配置中心
